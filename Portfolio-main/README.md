## 🚀 How to Customize

### 1. Add Your Profile Photo

- Place your photo in the `images/` folder
- Name it `profile.jpg` (or update the path in index.html)
- Recommended size: 400x400px or larger (square)

### 2. Update Your Name

In `index.html`, find:

```html
<span class="name-line">Your Name</span>
```

Replace "Your Name" with your actual name.

### 3. Change the Quote

In `index.html`, find:

```html
<p class="hero-quote">
  "The only way to do great work is to love what you do."
</p>
```

Replace with your favorite quote.

### 4. Update Navigation

The navigation links are placeholders. Create these pages:

- `academic.html` - Academic Success page
- `biography.html` - Biography page
- `projects.html` - Projects & Experience page
- `contact.html` - Contact page

## 🌐 How to Deploy

### Option 1: Vercel (Recommended - Easiest)

1. Install Vercel CLI: `npm install -g vercel`
2. In the portfolio folder, run: `vercel`
3. Follow the prompts
4. Connect your custom domain in Vercel dashboard

### Option 2: Netlify

1. Create account at netlify.com
2. Drag and drop your portfolio folder
3. Connect custom domain in settings

### Option 3: GitHub Pages

1. Create a GitHub repository
2. Upload all files
3. Go to Settings → Pages
4. Select main branch and save
5. Access at: `yourusername.github.io/repository-name`
